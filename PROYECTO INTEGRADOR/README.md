📚 Informe de Análisis: Relación entre las horas de estudio y la calificación obtenida en un examen
🎯 Objetivo
Analizar la relación entre las horas de estudio y la calificación obtenida en un examen mediante un modelo de regresión lineal simple, con el propósito de determinar si el tiempo dedicado al estudio influye en el rendimiento académico y puede utilizarse como una herramienta para realizar predicciones.

1. Descripción del conjunto de datos
La información fue obtenida mediante una encuesta diseñada y aplicada por Paola Cortes y Geraldine Prieto a estudiantes que recientemente presentaron un examen.

La base de datos está conformada por 50 registros y contiene información demográfica y académica de los participantes.

Las variables recopiladas son:

Edad
Sexo
Semestre actual
Horas de estudio
Calificación obtenida
Para el análisis de regresión lineal simple se utilizarán las siguientes variables:

Variable independiente (X): Horas de estudio.
Variable dependiente (Y): Calificación obtenida.
2. Análisis descriptivo
Como primera etapa del análisis se realizará una exploración de los datos para conocer el comportamiento de las variables recopiladas. Se obtendrán medidas descriptivas y representaciones gráficas que permitan identificar la distribución de las horas de estudio, las calificaciones y las características generales de la muestra.

Este análisis servirá como base para la construcción del modelo de regresión lineal.

3. Relación entre las horas de estudio y la calificación
Posteriormente se analizará la relación existente entre las horas de estudio y la calificación obtenida mediante un diagrama de dispersión y un modelo de regresión lineal simple.

Además de evaluar visualmente la tendencia entre ambas variables, se calcularán indicadores estadísticos que permitan determinar la intensidad de la relación y la capacidad predictiva del modelo.

4. Evaluación del modelo
Para verificar la calidad del modelo se analizarán distintos indicadores estadísticos, como el coeficiente de correlación, el coeficiente de determinación y el análisis de residuales.

Estos resultados permitirán determinar si la regresión lineal simple representa adecuadamente la relación entre las variables estudiadas y si puede emplearse para realizar estimaciones confiables.

Conclusiones
A partir del análisis estadístico será posible determinar si existe evidencia suficiente para afirmar que las horas de estudio influyen en la calificación obtenida en un examen.

Asimismo, se evaluará la utilidad del modelo de regresión lineal simple para explicar el rendimiento académico y generar recomendaciones basadas en los resultados obtenidos.

Recomendaciones
Con base en los resultados del análisis, se propondrán recomendaciones relacionadas con los hábitos de estudio y la planificación del tiempo, así como posibles mejoras para futuras investigaciones mediante la incorporación de variables adicionales que también influyen en el rendimiento académico.
